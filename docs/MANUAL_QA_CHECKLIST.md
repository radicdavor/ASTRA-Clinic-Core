# Manual QA Checklist

- Log in with an admin account.
- Review today's schedule.
- Create and update an appointment.
- Complete an appointment with material consumption.
- Confirm stock movements are created.
- Receive a purchase order line.
- Confirm inventory stock changed.
- Create a draft invoice from an appointment.
- Issue invoice and confirm fiscalization status is visible.
- Record partial or full payment.
- Create an API key and confirm raw key is shown only once.
- Deactivate an API key.
- Filter audit log for a changed entity.
