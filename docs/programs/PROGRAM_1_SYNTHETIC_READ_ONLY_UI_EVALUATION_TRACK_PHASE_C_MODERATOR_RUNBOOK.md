# Program 1 Phase C - Moderator Runbook

Status: ready for a separately authorized session.

## Before the Participant Arrives

1. Confirm the exact candidate commit SHA.
2. Confirm the working tree is clean and no unreviewed code is present.
3. Start the local synthetic demo stack.
4. Confirm `/health` returns healthy.
5. Log in with the repository-controlled demo account.
6. Open `/program1/synthetic-review`.
7. Confirm all five scenario IDs are present.
8. Confirm the persistent synthetic safety boundary is visible.
9. Confirm no real data is present anywhere on screen.
10. Close unrelated applications, notifications, patient systems, and documents.
11. Prepare a blank observation form using only the participant code.

Do not begin if any preflight check fails.

## Opening

1. Complete eligibility and consent.
2. Read the safety boundary aloud.
3. State: `We are evaluating the interface, not you.`
4. Ask the participant to think aloud, without sharing real clinical cases.
5. Explain that the moderator will not teach the interface during a task unless safety requires intervention.

## Moderation Rules

- use the task wording exactly on the first attempt
- do not point to the correct control
- allow up to two minutes per task before marking assistance needed
- record observable behavior before interpretation
- distinguish participant words from moderator inference
- never convert comments into clinical claims
- do not ask for patient examples
- stop immediately when a stop condition occurs

## Allowed Neutral Prompts

- `Što trenutno očekujete da će se dogoditi?`
- `Što vam ova oznaka znači?`
- `Gdje biste sljedeće pogledali?`
- `Možete li to reći bez primjera stvarnog pacijenta?`

## Closing

1. Ask the eight closing questions from the Phase B rubric.
2. Ask for the single highest-value improvement.
3. Confirm no real information was disclosed or retained.
4. Explain that feedback informs a later decision and does not authorize clinical use.
5. Complete the evidence and decision packet.
6. Remove temporary notes not included in the controlled session record.
