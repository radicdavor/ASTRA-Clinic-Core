# Program 1 Phase J9 - Demo Risk Register

| Risk | Mitigation | Talking Point | Blocker |
| --- | --- | --- | --- |
| Timeline is misunderstood as decision | repeat no-decision copy | timeline is source-linked context only | no |
| Finding is treated as diagnosis | show source and limitations | finding is not diagnosis | no |
| Viewer assumes production readiness | open with no-production disclaimer | demo candidate only | no |
| Accidental real-data entry | stop demo immediately | demo data only | yes |
| Permission confusion | explain role boundary | access is not authority | no |
| Audit noise confusion | explain audit is not Outcome Evidence | audit records access/actions | no |
| No-action UI misunderstood | show absence of action buttons | read-only by design | no |
| Missing source read as truth | use safe fallback | unlinked truth is no-go | no |
| Demo environment failure | use fallback plan | do not live-debug clinical claims | no |
