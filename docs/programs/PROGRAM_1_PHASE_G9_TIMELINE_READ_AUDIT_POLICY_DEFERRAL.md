# Program 1 Phase G9 - Timeline Read Audit Policy Deferral

Timeline successful read audit is deferred by default to avoid audit noise.

Policy:

- successful list read audit: deferred
- denied read audit: future candidate
- out-of-scope read audit: future candidate
- payload must be minimized
- no full clinical text in audit payload
- access audit is not Outcome Evidence

G9 does not implement audit writes.
