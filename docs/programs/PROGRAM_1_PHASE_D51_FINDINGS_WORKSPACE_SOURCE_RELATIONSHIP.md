# Program 1 Phase D51 - Findings Workspace Source Relationship

Status: source relationship contract

## Required Source Display

Read-only findings UI must show source context:

- source type
- source label
- source reference
- optional source document relation when available

## Safe Interpretation

Source-linked display means the finding can be traced to a source context. It does not mean the finding is a confirmed diagnosis, treatment plan or clinical decision.

## Forbidden Interpretation

The workspace must not imply:

- source proves diagnosis automatically
- source creates Task
- source creates Outcome Evidence
- source notifies patient
- source changes appointment status
