# Program 1 Synthetic Sandbox Maintenance Track Phase B5 - Scenario Expansion Decision Criteria

Status: documentation-only scenario expansion criteria. Synthetic-only. Non-production. Local terminal-only. Not for clinical use.

Phase B may ask whether more synthetic scenarios would help evaluation, but it does not add scenarios.

Questions before any future scenario expansion:

- What review question would a new synthetic scenario answer?
- Can alpha and beta answer the same question without adding scope?
- Would the scenario remain fully synthetic and non-clinical?
- Could the scenario be mistaken for real patient data or PHI/PII?
- Would the scenario imply diagnosis, treatment, triage, patient instruction, appointment action, or clinical recommendation?
- Would the scenario require runtime code changes?
- Would documentation-only clarification be enough?

Current decision: no scenario expansion is authorized by Phase B.
