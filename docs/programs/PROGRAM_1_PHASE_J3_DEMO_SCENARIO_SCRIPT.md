# Program 1 Phase J3 - Demo Scenario Script

## Scenario

1. Open a synthetic demo patient.
2. Explain the clinical document context as synthetic source material.
3. Show a source-linked finding and explain that it is not a diagnosis.
4. Show a source-linked open question and explain that it requires clinician interpretation.
5. Show readiness snapshot history as recorded demo context.
6. Show acknowledgment as human signal review, not approval or clearance.
7. Open Clinical Evidence Timeline and show how events aggregate source-linked records.
8. Explain governance and no-go boundaries.

## Required Talking Points

- no clinical decision is made
- no Task is created
- no patient message is sent
- no treatment is recommended automatically
- no appointment status is changed
- real patient data is not allowed
