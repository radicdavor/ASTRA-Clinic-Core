# Program 1 Phase H0 - Timeline Workspace Contract

The Clinical Evidence Timeline appears in the Patient Workspace as a patient-scoped read-only panel.

It relates to:

- findings as source-linked finding events
- open questions as unresolved review-context events
- readiness snapshots as advisory snapshot events
- acknowledgments as human-review signal events

Display rules:

- newest-first order from the API
- event label, event type, timestamp, provenance and limitations visible
- loading, empty, error and permission states use non-clinical wording
- no action buttons
- no diagnosis, treatment, approval, clearance or override wording
- no production or real-data approval
