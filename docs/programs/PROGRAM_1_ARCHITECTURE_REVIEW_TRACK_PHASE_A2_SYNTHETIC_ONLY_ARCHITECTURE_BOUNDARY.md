# Program 1 Architecture Review Track Phase A2 - Synthetic-Only Architecture Boundary

Architecture Review Track Phase A is synthetic-only.

## Allowed

- synthetic examples
- abstract entities
- non-patient data
- non-PHI placeholders
- non-PII placeholders
- documentation diagrams
- text-only models

## Prohibited

- real patient data
- live clinic data
- identifiable patient examples
- imported production datasets
- real appointment records
- real messages
- real clinical documents
- real audit events

Synthetic-only discussion is not real-data approval, PHI/PII approval, clinical deployment approval, or implementation authorization.
