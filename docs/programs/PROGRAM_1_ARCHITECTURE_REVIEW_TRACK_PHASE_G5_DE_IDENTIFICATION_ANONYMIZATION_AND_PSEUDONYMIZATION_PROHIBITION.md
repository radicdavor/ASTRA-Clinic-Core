# Program 1 Architecture Review Track Phase G5 - De-Identification, Anonymization, and Pseudonymization Prohibition

Status: documentation-only prohibition record.

## Conceptual topics only

De-identification, anonymization, and pseudonymization are future conceptual topics only.

Phase G does not implement:

- De-identification tooling.
- Anonymization tooling.
- Pseudonymization tooling.
- Pipelines.
- Transformations.
- Validators.
- Datasets.
- Claims that data is de-identified, anonymized, pseudonymized, or ready for use.

## Current decision

No de-identification, anonymization, or pseudonymization implementation is approved, cleared, implemented, validated, or authorized by Phase G.
