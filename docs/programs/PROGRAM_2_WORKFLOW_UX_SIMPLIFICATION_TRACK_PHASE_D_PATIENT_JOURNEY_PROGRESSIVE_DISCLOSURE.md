# Phase D — Progresivno otkrivanje tijeka pacijenta

Trostupčani prikaz zamijenjen je fokusiranim radnim prostorom: sažeto zaglavlje, najviše pet kritičnih činjenica, **Što sada treba napraviti?**, kompaktni stepper i samo jedna otvorena faza.

Faze su Dokumenti i priprema, Dolazak i prijem, Pregled, Materijal, Naplata i Završeno. Postojeći `focus` deep-link parametri ostaju podržani kroz kompatibilne alias-e. Mutacije se izvršavaju tek unutar otvorene faze, s postojećim potvrdama za kliničke i financijske posljedice.
