# Program 1 Architecture Review Track Phase I2 - Integration Conceptual Boundary

Status: documentation-only, synthetic-only, non-runtime integration boundary.

Integration is a future conceptual architecture domain only. Phase I2 may discuss future integration categories and dependencies but must not create runtime integration behavior.

## Prohibited current actions

- Data sync.
- Data import/export.
- Polling.
- Webhooks.
- Event subscriptions.
- Queues.
- Background jobs.
- Scheduled jobs.
- File ingestion.
- CSV imports.
- FHIR/HL7 integrations.
- Production system access.

## Decision

Integration remains conceptual and prohibited from implementation. Program 1 remains in pre-implementation hold.
