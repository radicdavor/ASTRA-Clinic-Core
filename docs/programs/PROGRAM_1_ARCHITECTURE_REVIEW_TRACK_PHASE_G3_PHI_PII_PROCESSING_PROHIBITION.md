# Program 1 Architecture Review Track Phase G3 - PHI/PII Processing Prohibition

Status: documentation-only PHI/PII prohibition record.

## Explicit prohibitions

Phase G prohibits:

- Real patient data ingestion.
- Real patient data processing.
- PHI/PII processing.
- Production dataset use.
- Imported clinic data.
- EHR/EMR extracts.
- Appointment data extracts.
- Patient messages.
- Clinical notes.
- Real audit logs.
- Real authorization logs.
- Patient-derived examples.
- Real-world case vignettes derived from identifiable or clinic data.

## Current decision

PHI/PII processing remains prohibited. Phase G does not approve, clear, implement, validate, or authorize any real-data or PHI/PII processing path.
