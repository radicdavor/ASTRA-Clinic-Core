# Program 1 Phase F14 - Timeline D/E Module Integration

Status: documented

## D Module

Findings lifecycle, findings persistence/read API/workspace, extraction candidates and open questions can map to passive timeline events.

## E Module

Review foundation can map review-pending or review-completed semantics, but not clinical decisions.

## Included as Timeline Context

ClinicalDocument, ClinicalFinding, ClinicalOpenQuestion, ClinicalFindingExtractionCandidate, ClinicalReviewPreview, readiness snapshots and acknowledgments.

## Excluded

Task, Outcome Evidence, patient messaging, approvals, clearances, overrides, automatic diagnosis/treatment and production governance claims.
