# Program 1 Phase E1 - Review Definition Boundary Contract

Status: documented

## Review Boundary

Review means human examination of source-linked clinical context. It can produce context for later decisions, but is not itself a clinical decision.

## Distinctions

- Acknowledgment: confirms a human reviewed a specific signal/context; not a review workflow.
- Physician decision: separate confirmed clinical decision.
- Recommendation: separate draft or confirmed recommendation.
- Finding: source-linked clinical information.
- Open question: unresolved source-linked question.
- Clinical document: source material.
- Task: operational work item; review is not a task.
- Outcome Evidence: proof/result object; review is not outcome evidence.
- Patient message: communication; review does not message patients.

## Explicit No-Go

Review is not diagnosis by itself, treatment plan, patient instruction, approval, clearance, override or automatic closure.
