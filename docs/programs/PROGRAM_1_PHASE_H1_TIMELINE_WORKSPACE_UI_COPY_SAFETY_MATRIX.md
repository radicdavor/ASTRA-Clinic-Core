# Program 1 Phase H1 - Timeline Workspace UI Copy Safety Matrix

Allowed labels:

- Klinicka vremenska crta
- Source-linked dogadjaji
- Povezano s izvorom
- Ceka pregled
- Potrebna odluka lijecnika
- Nije klinicka odluka
- Ne stvara zadatak
- Ne salje poruku pacijentu
- Ne mijenja status termina

Forbidden labels:

- Dijagnosticirano
- Terapija zapoceta
- Odobreno
- Clearance
- Override
- Rijeseno
- Task created
- Outcome confirmed
- Patient notified
- Automatski zakljuceno
