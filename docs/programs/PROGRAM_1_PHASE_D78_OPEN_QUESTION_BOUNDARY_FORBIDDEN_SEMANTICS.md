# Program 1 Phase D78 - Open Question Boundary and Forbidden Semantics

Status: forbidden semantics contract

## Allowed Language

- open question
- unresolved clinical question
- needs clinician interpretation
- source-linked question
- review needed
- not yet decided

## Forbidden Language

- task created
- outcome proven
- diagnosis confirmed
- treatment required
- patient notified
- resolved
- closed by AI
- approved
- cleared
- override
- final decision

## Boundary Rules

An open question is not a Task.

An open question is not diagnosis.

An open question is not recommendation.

An open question is not physician decision.

An open question is not patient communication.

An open question does not close a finding.

## Runtime Boundary

D78 does not add runtime behavior.
