# Program 1 Phase J13 - Demo Environment Checklist

- Windows or supported local development OS available
- Docker Desktop running
- backend `.env` uses demo/development settings
- no production secrets in demo
- PostgreSQL container healthy
- Alembic migrations can run
- frontend can reach backend
- browser can open the app
- ports 5173 and 8000, or documented alternate ports, are available
- demo seed/reset process is understood
- logs are available for backend and frontend
- stop criteria are understood
