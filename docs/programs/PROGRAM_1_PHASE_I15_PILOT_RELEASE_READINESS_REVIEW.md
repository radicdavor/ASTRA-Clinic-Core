# Program 1 Phase I15 - Pilot Release Readiness Review

## Allowed Pilot Shape

- closed demo/pilot
- demo data only
- controlled walkthrough with trained users
- explicit no-production and no-real-data messaging

## Can Be Shown

- readiness previews and snapshots
- acknowledgment history
- findings and open questions read-only views
- Clinical Evidence Timeline read API and workspace
- governance docs and known limitations

## Cannot Be Claimed

- production readiness
- real patient data approval
- certified EMR
- medical-device status
- automated diagnosis/treatment
- workflow enforcement
- clinical approval/clearance

## Pilot Checklist

- reset to demo data
- confirm users know real data is forbidden
- review roles and demo credentials
- explain no-decision semantics
- collect feedback on wording and usability
- keep fallback plan outside ASTRA for clinical work
