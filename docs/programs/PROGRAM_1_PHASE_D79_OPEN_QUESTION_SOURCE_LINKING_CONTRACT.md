# Program 1 Phase D79 - Open Question Source-Linking Contract

Status: source-linking contract

## Purpose

Define source traceability for open questions derived from findings.

## Required Source Links

Future open question previews must preserve:

- source finding reference when available
- source document reference when available
- source type
- source label
- source reference
- source text span if available
- extraction candidate reference if applicable
- confidence and limitations context

## Why Traceability Is Required

An unlinked question cannot be evaluated safely. It must remain draft/invalid and must not be displayed as official clinical truth.

## Review Boundary

Source linkage enables human review. It does not create diagnosis, recommendation, decision, task, outcome evidence or patient communication.
