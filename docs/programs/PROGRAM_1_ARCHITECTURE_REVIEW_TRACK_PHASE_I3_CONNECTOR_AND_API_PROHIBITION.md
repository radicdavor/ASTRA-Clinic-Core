# Program 1 Architecture Review Track Phase I3 - Connector and API Prohibition

Status: documentation-only, synthetic-only, non-runtime prohibition record.

Connectors and APIs are future conceptual topics only.

## Prohibited

- API clients.
- Database clients.
- EHR/EMR connectors.
- Patient portal connectors.
- Appointment system connectors.
- Messaging system connectors.
- External system adapters.
- Data connectors.
- Webhook receivers.
- Polling jobs.
- Queue consumers/producers.
- Runtime API endpoints for integration.

## Decision

No connector, API client, adapter, endpoint, import/export path, or runtime data exchange is approved by Phase I.
