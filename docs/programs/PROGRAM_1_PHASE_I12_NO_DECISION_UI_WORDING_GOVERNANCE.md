# Program 1 Phase I12 - No-Decision UI Wording Governance

## Forbidden Wording

- approved
- cleared
- clearance
- override
- resolved by AI
- diagnosis confirmed by system
- treatment started automatically
- task created
- outcome proven
- patient notified

## Allowed Wording

- source-linked
- needs review
- needs clinician decision
- not a clinical decision
- does not create task
- does not notify patient
- read-only

## Rule

UI wording must not turn a source-linked record, timeline event, finding, open question, review preview, snapshot or acknowledgment into a clinical decision.
