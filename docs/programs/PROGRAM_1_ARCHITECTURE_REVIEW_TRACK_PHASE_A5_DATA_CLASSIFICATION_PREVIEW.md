# Program 1 Architecture Review Track Phase A5 - Data Classification Preview

This is a conceptual classification preview only.

| Category | Allowed in Phase A? | Decision |
| --- | --- | --- |
| synthetic data | yes | allowed for documentation-only examples |
| placeholder data | yes | allowed if abstract and non-identifying |
| anonymized example data | no | prohibited because provenance and re-identification risk need future review |
| de-identified data | no | prohibited because future privacy/legal review is required |
| pseudonymized data | no | prohibited because future privacy/legal review is required |
| identifiable patient data | no | prohibited |
| PHI/PII | no | prohibited |
| production clinical data | no | prohibited |

Phase A permits only synthetic data and abstract placeholders. Real, identifiable, PHI/PII, production, or clinic-derived data remains prohibited.
