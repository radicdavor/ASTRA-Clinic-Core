# Program 1 Phase C1 - Enforcement Vocabulary and Forbidden Semantics

Status: vocabulary contract

## Purpose

This document locks the vocabulary for future Clinical Readiness work.

Language must keep ASTRA in advisory mode until maintainers explicitly approve otherwise.

## Allowed Vocabulary

Allowed terms:

- warning
- review required
- missing input
- unresolved readiness issue
- clinician review needed
- non-blocking signal
- advisory state
- demo/pilot only
- saved preview record
- source-linked warning

## Forbidden Vocabulary

Forbidden terms:

- approved
- cleared
- override
- pass/fail clearance
- procedure approved
- patient ready for procedure
- automatic block
- automatic reschedule
- medically certified
- clinical decision
- final clearance

## Croatian Preferred Wording

Safe Croatian wording:

- upozorenje
- potreban pregled
- nedostaje podatak
- nerijeseno readiness pitanje
- potreban klinicki pregled
- neblokirajuci signal
- savjetodavno stanje
- demo/pilot only

Forbidden Croatian wording:

- odobreno
- propusnica
- pacijent je spreman
- postupak je odobren
- automatska blokada
- konacna klinicka odluka

## Enforcement Boundary

Vocabulary must never imply that ASTRA has made a clinical decision.
