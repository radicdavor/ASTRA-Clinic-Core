# Program 1 Phase E2 - Review Object Scope Source-Linking Contract

Status: documented

## Reviewable Object Scope

- ClinicalDocument review.
- Finding review.
- Open Question review.
- Extraction Candidate review.
- Source-linked evidence review.

## Source-Linking Rules

Every review concept must preserve source type, source label and source reference. If a source is unreviewed, review wording must say so. Source references are not rewritten by review.

## Immutability and Limitations

Review must not mutate source documents, findings, open questions or extraction candidates. Limitations must remain visible so review does not become official clinical truth by display alone.
