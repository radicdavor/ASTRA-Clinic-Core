# Controlled clinic pilot — usability metrics and decision thresholds

These thresholds are prospective. **No threshold is marked achieved because no human evidence has been supplied.**

| Metric | Required threshold |
|---|---|
| Non-clinical administrative task success | ≥ 90% |
| Wrong-patient consequential actions | 0 |
| Missed hard blockers | 0 |
| Unauthorized clinical decisions | 0 |
| Median next-action identification | ≤ 5 seconds |
| Task completion | Within task-specific time in Phase D |
| Median confidence | ≥ 4/5 |
| Median perceived effort | ≤ 2/5 |
| Prohibited primary action visible to a role | 0 |
| Need to know internal module names | 0 participants |
| Automatic AI diagnosis insertion | 0 |
| Duplicate invoice/payment | 0 |

Decision requires at least one representative participant per operational role, completed forms, defect/safety-event log, build identity and moderator sign-off. Any wrong-patient action, missed hard blocker, unauthorized clinical action, automatic AI writeback or duplicate financial mutation overrides aggregate percentages and yields **NOT READY — BLOCKERS REMAIN** until a separately scoped fix and repeat evaluation.
