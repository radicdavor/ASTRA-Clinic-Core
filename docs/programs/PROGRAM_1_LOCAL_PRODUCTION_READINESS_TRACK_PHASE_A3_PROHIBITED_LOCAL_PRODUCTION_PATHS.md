# Program 1 Local Production Readiness Track Phase A3 - Prohibited Local Production Paths

Status: prohibited paths register. Documentation-only.

The following paths are prohibited by Phase A:

- local production with real patient data
- local production with PHI/PII
- local production connected to EHR/EMR
- local production connected to appointment systems
- local production connected to patient messaging
- local production with database persistence
- local production with exports
- local production with staff workflow enforcement
- local production with clinical writeback
- local production with diagnosis/treatment execution
- local production with autonomous triage
- local production with approval/override capability
- local production claiming go-live or clinical readiness

Current decision: none of these paths are approved or implemented.
