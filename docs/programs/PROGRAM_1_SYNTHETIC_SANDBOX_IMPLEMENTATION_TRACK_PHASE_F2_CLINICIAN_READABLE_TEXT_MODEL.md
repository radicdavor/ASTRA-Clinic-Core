# Program 1 Synthetic Sandbox Implementation Track Phase F2 - Clinician-Readable Text Model

Synthetic-only. Non-production. No real patient data. No PHI/PII. Not for clinical use.

## Text Model

Default terminal output now uses these sections:

- Program 1 Synthetic Sandbox status banner
- Scenario
- Patient
- Encounter
- Findings
- Clinician review note
- Safety confirmations

The walkthrough command also includes:

- what the walkthrough demonstrates
- available synthetic scenarios
- clinician-readable scenario examples
- feedback demonstration
- design iteration queue explanation
- suggested local commands
- recommended clinician checklist

## JSON Boundary

JSON output may retain structured keys and internal synthetic identifiers. JSON remains synthetic-only and includes safety flags. JSON does not imply production-readiness, clinical use authorization, real-data use, PHI/PII handling or go-live.
