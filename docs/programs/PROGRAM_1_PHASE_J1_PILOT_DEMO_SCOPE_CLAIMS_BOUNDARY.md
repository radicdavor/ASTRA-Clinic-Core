# Program 1 Phase J1 - Pilot Demo Scope Claims Boundary

## Allowed Claims

- demo-only
- source-linked clinical workflow foundation
- read-only timeline
- findings/open questions read surfaces
- readiness snapshot governance
- acknowledgment governance
- no-decision support
- no patient messaging
- no production use

## Forbidden Claims

- production ready
- real patient data ready
- certified EMR
- certified medical device
- automated diagnosis
- automated treatment
- clinical approval engine
- clearance engine
- Outcome Evidence system
- Task engine
- patient communication platform

## Speaker Rule

Every demo introduction must say that ASTRA is a closed demo/pilot candidate with synthetic data only.
